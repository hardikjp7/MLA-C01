🎓 Exam Preparation — Format, Strategy & Resources
📝 Exam Format (MLA-C01)
Duration170 minutes (3 hrs)
Questions65 (85 during beta)
Cost$300 USD
Passing Score~720/1000
Pace target~2.5 min/question
Question Types:
Multiple Choice
1 correct answer out of 4. Classic format.
Multiple Response
2+ correct answers out of 5+. No partial credit.
Ordering ★ NEW
Put 3-5 responses in the correct order. No partial credit.
Matching ★ NEW
Match responses to 3-7 prompts. No partial credit.
Case Study ★ NEW
Same scenario across 2+ questions. Each evaluated separately.
⚡ No partial credit on Ordering, Matching questions — either all right or all wrong. Prioritize Multiple Choice/Response if pressed for time.
🧠 Exam Strategy
Before the Exam:
  • Use bathroom before heading in
  • Eat something (but not right before)
  • Get a good night's sleep
  • Review must-memorize facts (Recall, Precision, F1, Regularization)
  • Schedule when your energy is highest
  • Practice exams repeatedly until consistent scores
During the Exam:
  • Don't rush — understand what you're optimizing for
  • For each question: understand requirements + the whole system
  • Target 2–2.5 minutes per question
  • Taking too long? Best guess → flag → revisit later
  • Use the flag capability! You'll have time to go back.
  • Process of elimination — eliminate wrong answers first
  • Note paper/dry-erase board provided. Use it for formulas!
  • Keep calm — you don't need 100%
Question Mindset:
  • What metric/outcome is being optimized for?
  • What are the specific requirements/constraints?
  • What is the BEST answer (not just a correct one)?
  • Think about cost, scalability, serverless vs managed
Resources: AWS Skill Builder (free "Standard Exam Prep Plan" + 20-question practice set). SageMaker Developer Guide. Amazon's official Exam Guide. Practice exams on Udemy/Whizlabs.
⚡ Last-Minute Must-Memorize
Metrics Formulas:
Recall = TP / (TP + FN)
Precision = TP / (TP + FP)
F1 = 2 × (P × R) / (P + R)
Specificity = TN / (TN + FP)
AUC 0.5 = useless | AUC 1.0 = perfect
Key Numbers:
S3 durability: 11 nines (99.999999999%)
Kinesis shard: 1MB/s in, 2MB/s out
Ground Truth: 70% cost reduction
Managed Spot: up to 90% cost savings
Async inference: up to 1GB payload
Inference Recommender: 45min / 2hr
Shield Advanced: $3,000/month/org
Step Functions max: 1 year execution
Trainium: 50% training cost savings
Inferentia: 70% inference cost savings, 4x throughput
DistilBERT: 40% smaller than BERT
EMR Serverless: 10% memory overhead for Spark
When to use Recall vs Precision:
Recall: FN is costly → fraud, disease detection
Precision: FP is costly → drug tests, spam filters
F1: Both matter → balanced approach
